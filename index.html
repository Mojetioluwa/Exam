        6.  Explain how comments in code works and give an example of using comments in html (5 Marks)

A comment is a programmer-readable explanation or annotation in the source code of a computer program. They are added with the purpose of making the source code easier for humans to understand, and are generally ignored by compilers and interpreters. Comments serves like a pointer to your code explaining what you do in each section of the code making it easy to understand to read.
